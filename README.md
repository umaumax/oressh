# oressh

oreore ssh command

## NOTE
* ssh先の環境を汚さずにoreore bashとoreore vimの設定を利用する
* functionやaliasではsshpassから使用できないため，コマンドとして提供

### for wgit
```
https://github.com/umaumax/oressh/blob/master/oressh.sh oressh
chmod u+x oressh
```

## Required
* [umaumax/dotfiles]( https://github.com/umaumax/dotfiles )
